Read Me:

Complie and Run the program:


1.Download the zip file and unpack
2.Ensure that the Questions.txt and HighScore.txt are in the same folder as the java files.
3.Open the terminal from within the folder
4.Compile the code by typing "javac Main.java" and press enter
5.Run the code by typoing "java Main" and press enter

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 

Playing the Game:

1. Enter players name.
2. Pick a category and point value.
3. Read the question and choose the choice you think is correct.
4. A dialog box will notify you if your answer is correct or incorrect and add or take away points depending on the outcome.
5. Keep going until all options are answered.
6. The player with the most points after everything has been answered, wins :)!

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 

Problem Solving

1. If the Question file is not loading ensure the file is in the same folder as the program files
2. If the HighScore file is not loading ensure the file is in the same folder as the program files
